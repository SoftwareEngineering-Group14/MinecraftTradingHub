# MicraftTradingHub

## Overview
This project is for Software Engineering (001), Spring Semester of 2026.

The goal of this project is to create a web app that will serve as a trading platform for Minecraft Servers.  This will allow players ti create an interactive storefront for trading with their friends by displaying what in-game items they would like to trade, and what they would like in exchange.
